The project is a simple search bar that filters users based on search input. The project is made with html, css and javascript. To build this project, first a structure is created with html that has a search bar and user cards containing the name and email of users. The search bar and user cards are then styled with css. Then using an api, dummy user information is fetched from an external website and finally with the help of javascript, users are filtered by matching the search input.
